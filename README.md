# Gray
Gray는 암호화된 알려진 단어의 데이터베이스를 구축하는 프로젝트입니다.

## 작업내용
1. 100만개 이상의 데이터 수집완료
2. SHA-256, SHA3-256 데이터 수집 완료

## 효과
데이터에 알려진 password 데이터도 상당수 존재하므로 알려진 암호 사용을 막을 수 있습니다.
